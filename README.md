# currency-rate-internship-task

Clone repository <br />
```
git clone https://github.com/tkajda/currency-rate-internship-task
```

___

### Run backend 
```
cd ./backend
```

```
mvn spring-boot:run
```

___

### Run frontend <br />
```
cd ./frontend
```

```
npm install
```

```
http-server
```

___



### Tests:<br />
```
npm test
```
